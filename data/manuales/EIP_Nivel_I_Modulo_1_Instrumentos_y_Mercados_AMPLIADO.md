## 1. FACTORES MACROECONÓMICOS

### 1.1 Fundamentos Macroeconómicos

#### A) Indicadores de Crecimiento Económico

**PIB (Producto Interior Bruto)**
- **Definición:** Valor monetario de todos los bienes y servicios finales producidos en un país durante un período.
- **PIB nominal:** A precios corrientes (incluye inflación)
- **PIB real:** A precios constantes (elimina efecto inflación)

**Fórmula del PIB:**
```
PIB = C + I + G + (X - M)

Donde:
C = Consumo privado
I = Inversión
G = Gasto público
X = Exportaciones
M = Importaciones
```

**Ejemplo numérico:**
```
Si en un país:
- Consumo = 600.000 M€
- Inversión = 200.000 M€
- Gasto público = 250.000 M€
- Exportaciones = 300.000 M€
- Importaciones = 350.000 M€

PIB = 600.000 + 200.000 + 250.000 + (300.000 - 350.000) = 1.000.000 M€
```

**Tasa de crecimiento del PIB:**
```
Crecimiento PIB = [(PIB año actual - PIB año anterior) / PIB año anterior] × 100
```

**Otros indicadores de crecimiento:**
- **Producción industrial:** Mide la actividad del sector manufacturero
- **PMI (Purchasing Managers Index):** >50 = expansión; <50 = contracción
- **Ventas minoristas:** Indicador de consumo

#### B) Indicadores de Precios (Inflación)

**IPC (Índice de Precios al Consumo)**
- Mide la variación de precios de una cesta de bienes y servicios
- Es el indicador principal de inflación

**IPC Subyacente**
- Excluye alimentos no elaborados y energía (más volátiles)
- Muestra la tendencia de fondo de la inflación

**Deflactor del PIB**
```
Deflactor = (PIB nominal / PIB real) × 100
```

**Ejemplo numérico:**
```
Si PIB nominal = 1.100.000 M€ y PIB real = 1.000.000 M€
Deflactor = (1.100.000 / 1.000.000) × 100 = 110
Inflación implícita = 10%
```

**Tipo de interés real vs nominal (Ecuación de Fisher):**
```
(1 + i nominal) = (1 + i real) × (1 + inflación)

Aproximación: i real ≈ i nominal - inflación
```

**Ejemplo:**
```
Si tipo nominal = 5% e inflación = 3%
Tipo real exacto: (1,05/1,03) - 1 = 1,94%
Aproximación: 5% - 3% = 2%
```

#### C) Indicadores de Empleo

- **Tasa de paro:** Desempleados / Población activa × 100
- **Tasa de actividad:** Población activa / Población en edad de trabajar × 100
- **Creación de empleo:** Variación neta de ocupados

### 1.2 Los Ciclos Económicos

**Definición:** Fluctuaciones recurrentes de la actividad económica alrededor de su tendencia a largo plazo.

**Fases del ciclo:**

| Fase | Características | Efecto típico en activos |
|------|-----------------|-------------------------|
| **Expansión** | PIB↑, Empleo↑, Consumo↑ | RV↑, RF estable |
| **Auge (Pico)** | Máximo crecimiento, Inflación↑ | RV↑ pero riesgo alto |
| **Recesión** | PIB↓, Empleo↓, Consumo↓ | RV↓, RF↑ (flight to quality) |
| **Depresión (Valle)** | Mínimo de actividad | RF↑, RV en mínimos |

**Variables procíclicas y anticíclicas:**
- **Procíclicas:** Se mueven en la misma dirección que el PIB (empleo, consumo, inversión)
- **Anticíclicas:** Se mueven en dirección opuesta (tasa de paro)

**Crecimiento tendencial:**
```
Tasa media anual = [(Valor final / Valor inicial)^(1/n)] - 1
```

**Ejemplo:**
```
Si PIB pasa de 1.000.000 M€ a 1.200.000 M€ en 5 años:
Tasa media = (1.200.000/1.000.000)^(1/5) - 1 = 3,71% anual
```

### 1.3 Indicadores de Coyuntura

**Clasificación:**

| Tipo | Descripción | Ejemplos |
|------|-------------|----------|
| **Adelantados** | Anticipan el ciclo | PMI, Confianza consumidor, Pedidos industriales |
| **Coincidentes** | Se mueven con el ciclo | PIB, Producción industrial |
| **Retardados** | Confirman el ciclo | Tasa de paro, Morosidad |

**Principales indicadores por región:**

| Región | Indicadores clave |
|--------|-------------------|
| **EE.UU.** | Non-Farm Payrolls, ISM, Consumer Confidence |
| **Europa** | PMI Eurozona, IFO (Alemania), Sentix |
| **China** | PMI Caixin, Producción industrial |

### 1.4 El BCE y la Política Monetaria

**Estructura del Eurosistema:**
- BCE (Banco Central Europeo)
- Bancos centrales nacionales de la zona euro

**Objetivo principal:** Estabilidad de precios (inflación cercana al 2%)

**Diferencia con la Fed (EE.UU.):**
- BCE: Mandato único (inflación)
- Fed: Mandato dual (inflación + empleo)

**Instrumentos de política monetaria:**

| Instrumento | Descripción |
|-------------|-------------|
| **Operaciones de mercado abierto** | Compra/venta de activos para inyectar/drenar liquidez |
| **Facilidades permanentes** | Facilidad de crédito y de depósito |
| **Reservas mínimas** | Coeficiente de caja obligatorio |

**Tipos oficiales del BCE:**
- **Tipo de refinanciación principal (MRO):** Tipo central
- **Facilidad de depósito:** Suelo del corredor
- **Facilidad de crédito:** Techo del corredor

**Regla de Taylor (concepto):**
```
Tipo óptimo = Tipo neutral + 0,5×(Inflación - Objetivo) + 0,5×(PIB real - PIB potencial)
```

### 1.5 Tipos de Interés de Referencia

**€STR (Euro Short-Term Rate):**
- Sustituye al EONIA desde 2022
- Tipo a un día del mercado monetario no garantizado

**EURIBOR (Euro Interbank Offered Rate):**
- Plazos: 1 semana, 1, 3, 6 y 12 meses
- Referencia para hipotecas y préstamos

**Curva de tipos:**
- **Normal/creciente:** Tipos a largo > tipos a corto (expectativa de crecimiento)
- **Invertida:** Tipos a corto > tipos a largo (expectativa de recesión)
- **Plana:** Tipos similares en todos los plazos

### 1.6 Factores ESG y Sostenibilidad

**Definición ESG:**
- **E (Environmental):** Cambio climático, emisiones, recursos naturales
- **S (Social):** Condiciones laborales, diversidad, comunidad
- **G (Governance):** Estructura directiva, ética, transparencia

**Marco regulatorio europeo:**

| Normativa | Contenido |
|-----------|-----------|
| **Taxonomía UE** | Define qué actividades son "sostenibles" |
| **SFDR** | Divulgación de sostenibilidad en servicios financieros |
| **Pacto Verde Europeo** | Objetivo: neutralidad climática 2050 |

**Principio DNSH (Do No Significant Harm):**
Una actividad es sostenible si contribuye a un objetivo medioambiental SIN perjudicar significativamente a los otros cinco.

**Los 6 objetivos medioambientales de la Taxonomía:**
1. Mitigación del cambio climático
2. Adaptación al cambio climático
3. Uso sostenible del agua
4. Economía circular
5. Prevención de la contaminación
6. Biodiversidad y ecosistemas

## 2. FUNDAMENTOS DE LA INVERSIÓN (Matemáticas Financieras)

### 2.1 Valor Temporal del Dinero

**Principio fundamental:** Un euro hoy vale más que un euro mañana.

**Conceptos básicos:**
- **Capital financiero:** (C, t) donde C = cuantía y t = momento
- **Capitalización:** Llevar un capital del presente al futuro
- **Actualización (Descuento):** Traer un capital del futuro al presente

### 2.2 Capitalización Simple

**Fórmula:**
```
Cn = C0 × (1 + i × n)

Donde:
Cn = Capital final
C0 = Capital inicial
i = Tipo de interés (en tanto por uno)
n = Número de períodos
```

**Ejemplo 1: Calcular capital final**
```
Invertimos 10.000€ al 3% anual simple durante 2 años.

Cn = 10.000 × (1 + 0,03 × 2) = 10.000 × 1,06 = 10.600€
Intereses = 600€
```

**Ejemplo 2: Calcular tipo de interés**
```
Si C0 = 5.000€, Cn = 5.300€, n = 1,5 años. ¿Cuál es el tipo?

5.300 = 5.000 × (1 + i × 1,5)
1,06 = 1 + 1,5i
i = 0,06/1,5 = 0,04 = 4% anual
```

**Interés simple anticipado:**
```
C0 = Cn × (1 - d × n)

Donde d = tipo de descuento anticipado
```

**Relación entre interés vencido (i) y anticipado (d):**
```
i = d / (1 - d×n)
d = i / (1 + i×n)
```

### 2.3 Capitalización Compuesta

**Fórmula:**
```
Cn = C0 × (1 + i)^n
```

**Ejemplo 1: Capital final**
```
Invertimos 10.000€ al 5% anual compuesto durante 3 años.

Cn = 10.000 × (1,05)^3 = 10.000 × 1,157625 = 11.576,25€
```

**Ejemplo 2: Calcular el tipo de interés**
```
Si C0 = 8.000€, Cn = 10.000€, n = 4 años

10.000 = 8.000 × (1+i)^4
(1+i)^4 = 1,25
1+i = 1,25^(1/4) = 1,0574
i = 5,74% anual
```

**Ejemplo 3: Calcular el plazo**
```
Si C0 = 15.000€, Cn = 20.000€, i = 6%

20.000 = 15.000 × (1,06)^n
1,333 = 1,06^n
n = ln(1,333)/ln(1,06) = 4,93 años
```

**Capitalización con tipo variable:**
```
Cn = C0 × (1+i1) × (1+i2) × ... × (1+in)
```

**Ejemplo:**
```
C0 = 10.000€. Año 1: 3%, Año 2: 4%, Año 3: 5%

Cn = 10.000 × 1,03 × 1,04 × 1,05 = 11.248,20€
```

### 2.4 Descuento

#### Descuento Simple Comercial
```
E = N × (1 - d × n)
D = N × d × n

Donde:
E = Efectivo (valor actual)
N = Nominal (valor futuro)
d = Tipo de descuento
D = Descuento
```

**Ejemplo:**
```
Descontamos un pagaré de 5.000€ a 90 días al 6% anual.
n = 90/360 = 0,25 años

E = 5.000 × (1 - 0,06 × 0,25) = 5.000 × 0,985 = 4.925€
Descuento = 75€
```

#### Descuento Simple Racional (Matemático)
```
E = N / (1 + i × n)
```

**Ejemplo:**
```
Mismo pagaré al 6% racional:
E = 5.000 / (1 + 0,06 × 0,25) = 5.000 / 1,015 = 4.926,11€
```

#### Descuento Compuesto
```
C0 = Cn / (1+i)^n = Cn × (1+i)^(-n)
```

### 2.5 Tipos de Interés Spot y Forward

**Tipo Spot:** Tipo de interés desde hoy hasta un momento futuro.

**Tipo Forward:** Tipo de interés implícito entre dos momentos futuros.

**Fórmula para calcular el forward:**
```
(1 + r0,n)^n = (1 + r0,m)^m × (1 + fm,n)^(n-m)

Donde:
r0,n = tipo spot a n períodos
r0,m = tipo spot a m períodos
fm,n = tipo forward entre m y n
```

**Ejemplo práctico:**
```
Datos:
- Tipo spot 1 año: 2%
- Tipo spot 2 años: 3%

¿Cuál es el forward del año 1 al año 2?

(1,03)^2 = (1,02)^1 × (1 + f1,2)^1
1,0609 = 1,02 × (1 + f1,2)
f1,2 = 1,0609/1,02 - 1 = 4,01%
```

**Interpretación:** El mercado espera que el tipo a 1 año dentro de un año sea del 4,01%.

### 2.6 Medidas de Rentabilidad

#### Rentabilidad Simple
```
Rentabilidad = (Valor final - Valor inicial + Rentas) / Valor inicial × 100
```

**Ejemplo:**
```
Compra acción: 50€
Venta acción: 55€
Dividendo: 2€

Rentabilidad = (55 - 50 + 2) / 50 × 100 = 14%
```

#### Rentabilidad Real
```
Rentabilidad real = [(1 + Rent. nominal) / (1 + Inflación)] - 1
```

**Ejemplo:**
```
Rentabilidad nominal = 8%, Inflación = 3%

Rentabilidad real = (1,08/1,03) - 1 = 4,85%
```

#### TAE (Tasa Anual Equivalente)
```
TAE = (1 + i/m)^m - 1

Donde m = número de capitalizaciones al año
```

**Ejemplo:**
```
Tipo nominal 6% con capitalización mensual:
TAE = (1 + 0,06/12)^12 - 1 = (1,005)^12 - 1 = 6,17%
```

#### TIR (Tasa Interna de Rentabilidad)

**Definición:** Tipo de interés que iguala el valor actual de los flujos de caja a cero.

```
0 = -Inversión + Σ [Flujo_t / (1+TIR)^t]
```

**Ejemplo simplificado:**
```
Inversión: 1.000€
Flujo año 1: 600€
Flujo año 2: 550€

0 = -1.000 + 600/(1+TIR) + 550/(1+TIR)^2

Resolviendo: TIR ≈ 10%
```

#### TGR (Tasa Geométrica de Rentabilidad)
```
TGR = [(1+r1) × (1+r2) × ... × (1+rn)]^(1/n) - 1
```

**Ejemplo:**
```
Año 1: +20%, Año 2: -10%, Año 3: +15%

TGR = [(1,20) × (0,90) × (1,15)]^(1/3) - 1
TGR = [1,242]^(1/3) - 1 = 7,49%
```

### 2.7 Rentas Financieras

**Definición:** Conjunto de capitales con vencimientos periódicos.

**Clasificación:**
- Por duración: Temporales / Perpetuas
- Por momento de pago: Prepagables (inicio) / Postpagables (final)
- Por inicio: Inmediatas / Diferidas
- Por cuantía: Constantes / Variables

#### Renta Constante, Temporal, Inmediata, Postpagable

**Valor Actual:**
```
VA = C × [(1 - (1+i)^(-n)) / i]
```

**Valor Final:**
```
VF = C × [((1+i)^n - 1) / i]
```

**Ejemplo:**
```
Renta de 1.000€ anuales durante 5 años al 4%

VA = 1.000 × [(1 - 1,04^(-5)) / 0,04]
VA = 1.000 × [(1 - 0,8219) / 0,04]
VA = 1.000 × 4,4518 = 4.451,82€

VF = 1.000 × [(1,04^5 - 1) / 0,04]
VF = 1.000 × [(1,2167 - 1) / 0,04]
VF = 1.000 × 5,4163 = 5.416,32€
```

#### Renta Prepagable (pagos al inicio del período)

```
VA prepagable = VA postpagable × (1+i)
VF prepagable = VF postpagable × (1+i)
```

#### Renta Perpetua Postpagable

```
VA = C / i
```

**Ejemplo:**
```
Renta perpetua de 500€ anuales al 5%
VA = 500 / 0,05 = 10.000€
```

#### Renta Diferida

Si la renta comienza dentro de d períodos:
```
VA diferida = VA inmediata × (1+i)^(-d)
```

### 2.8 Conceptos Básicos de Estadística

#### Media Aritmética
```
x̄ = Σxi / n
```

#### Esperanza Matemática (datos con probabilidades)
```
E(x) = Σ(xi × pi)
```

**Ejemplo:**
```
Escenario | Rentabilidad | Probabilidad
Bueno     | 20%          | 30%
Normal    | 8%           | 50%
Malo      | -5%          | 20%

E(r) = 0,20×0,30 + 0,08×0,50 + (-0,05)×0,20
E(r) = 0,06 + 0,04 - 0,01 = 9%
```

#### Varianza y Desviación Típica
```
Varianza (σ²) = Σ[(xi - x̄)² × pi]
Desviación típica (σ) = √Varianza
```

**Ejemplo (continuación):**
```
σ² = (0,20-0,09)²×0,30 + (0,08-0,09)²×0,50 + (-0,05-0,09)²×0,20
σ² = 0,0121×0,30 + 0,0001×0,50 + 0,0196×0,20
σ² = 0,00363 + 0,00005 + 0,00392 = 0,0076
σ = √0,0076 = 8,72%
```

#### Covarianza
```
Cov(x,y) = Σ[(xi - x̄)(yi - ȳ) × pi]
```

- Cov > 0: Se mueven en la misma dirección
- Cov < 0: Se mueven en direcciones opuestas
- Cov = 0: No hay relación lineal

#### Coeficiente de Correlación
```
ρ = Cov(x,y) / (σx × σy)
```

- ρ = +1: Correlación positiva perfecta
- ρ = -1: Correlación negativa perfecta
- ρ = 0: Sin correlación lineal

#### Recta de Regresión (Mínimos Cuadrados)
```
y = α + β×x

β = Cov(x,y) / Var(x)
α = ȳ - β×x̄
```

#### Coeficiente de Determinación (R²)
```
R² = ρ²
```

Indica el porcentaje de variación de Y explicado por X.

## 3. SISTEMA FINANCIERO

### 3.1 Componentes del Sistema Financiero

**Función principal:** Canalizar el ahorro hacia la inversión productiva.

**Componentes:**

| Componente | Descripción |
|------------|-------------|
| **Activos financieros** | Instrumentos que representan derechos (acciones, bonos, etc.) |
| **Intermediarios** | Entidades que conectan ahorradores e inversores |
| **Mercados** | Lugares/sistemas donde se negocian los activos |
| **Reguladores** | Organismos que supervisan el sistema |

### 3.2 Intermediarios Financieros

**Clasificación:**

| Tipo | Función | Ejemplos |
|------|---------|----------|
| **Bancarios** | Transforman activos (captan depósitos, dan créditos) | Bancos, Cajas |
| **No bancarios** | Median sin transformar | Brokers, Agencias de valores |
| **Inversión colectiva** | Gestionan patrimonios | Fondos de inversión, Pensiones |
| **Aseguradoras** | Gestionan riesgos | Compañías de seguros |

### 3.3 Activos Financieros

**Características principales:**

| Característica | Descripción |
|----------------|-------------|
| **Rentabilidad** | Rendimiento esperado de la inversión |
| **Riesgo** | Incertidumbre sobre el rendimiento |
| **Liquidez** | Facilidad para convertir en efectivo sin pérdida |

**Relación:** Mayor rentabilidad esperada → Mayor riesgo asumido

### 3.4 Mercados Financieros

**Clasificación por fase:**
- **Mercado primario:** Emisión inicial de valores (el emisor capta fondos)
- **Mercado secundario:** Negociación posterior entre inversores

**Clasificación por organización:**
- **Mercados organizados:** Reglas, transparencia, supervisión (Bolsas)
- **Mercados OTC:** Negociación bilateral, menos transparencia

**Funciones de los mercados:**
1. Facilitar la liquidez
2. Descubrimiento de precios
3. Reducir costes de transacción
4. Canalizar información

### 3.5 Mercados Monetarios

**Características:**
- Activos a corto plazo (< 18 meses)
- Alta liquidez
- Bajo riesgo

**Activos del mercado monetario:**

| Activo | Emisor | Plazo |
|--------|--------|-------|
| Letras del Tesoro | Estado | 3, 6, 9, 12 meses |
| Pagarés de empresa | Empresas | Hasta 18 meses |
| Repos | Varios | Muy corto plazo |

## 4. MERCADO DE RENTA FIJA

### 4.1 Características Generales

**Definición:** Activos que generan flujos de caja predeterminados (cupones y principal).

> ⚠️ **Importante:** "Renta fija" NO significa precio fijo. El precio varía con los tipos de interés.

**Componentes de un bono:**

| Elemento | Descripción |
|----------|-------------|
| **Valor nominal** | Principal que se devuelve al vencimiento |
| **Cupón** | Interés periódico (fijo o variable) |
| **Precio** | Valor de mercado del bono |
| **Vencimiento** | Fecha de devolución del principal |
| **TIR (YTM)** | Rentabilidad total si se mantiene hasta vencimiento |

### 4.2 Mercado de Deuda Pública Española

**Emisores:**
- Tesoro Público (Estado)
- Comunidades Autónomas
- Entidades locales

**Tipos de activos:**

| Activo | Plazo | Cupón |
|--------|-------|-------|
| **Letras del Tesoro** | 3, 6, 9, 12 meses | Sin cupón (descuento) |
| **Bonos del Estado** | 3 y 5 años | Cupón anual |
| **Obligaciones del Estado** | 10, 15, 30, 50 años | Cupón anual |

**Miembros del mercado:**
- **Tesoro Público:** Emisor
- **Banco de España:** Agente financiero del Tesoro
- **Creadores de Mercado:** Obligados a dar liquidez
- **Entidades gestoras:** Custodios de valores

**Subasta del Tesoro:**
- **Peticiones competitivas:** Indican precio
- **Peticiones no competitivas:** Aceptan precio resultante
- **Precio marginal:** Precio más bajo aceptado
- **Prorrateo:** Si hay exceso de demanda al precio marginal

### 4.3 Mercado de Renta Fija Privada

**Tipos de activos:**

| Activo | Características |
|--------|-----------------|
| **Pagarés** | Corto plazo, descuento |
| **Bonos corporativos** | Medio/largo plazo, cupón |
| **Cédulas hipotecarias** | Respaldadas por préstamos hipotecarios |
| **Bonos convertibles** | Convertibles en acciones |
| **Participaciones preferentes** | Híbridos (deuda subordinada) |
| **Bonos verdes** | Financian proyectos medioambientales |
| **Bonos sostenibles** | Financian proyectos ESG |

### 4.4 Rating (Calificación Crediticia)

**Agencias principales:** S&P, Moody's, Fitch

**Escala de rating:**

| Categoría | S&P/Fitch | Moody's | Significado |
|-----------|-----------|---------|-------------|
| **Investment Grade** | AAA | Aaa | Máxima calidad |
| | AA | Aa | Muy alta calidad |
| | A | A | Alta calidad |
| | BBB | Baa | Calidad media |
| **High Yield (Bonos basura)** | BB | Ba | Especulativo |
| | B | B | Altamente especulativo |
| | CCC-C | Caa-C | Riesgo muy alto |
| | D | D | En impago |

### 4.5 Riesgos de la Renta Fija

| Riesgo | Descripción |
|--------|-------------|
| **De tipo de interés** | Variación del precio por cambios en tipos |
| **De crédito** | Posibilidad de impago del emisor |
| **De reinversión** | Reinvertir cupones a tipos menores |
| **De liquidez** | Dificultad para vender sin impactar precio |
| **De inflación** | Erosión del poder adquisitivo |
| **De tipo de cambio** | Si el bono está en otra divisa |
| **De amortización anticipada** | Bonos callable |

### 4.6 Valoración de Renta Fija

#### Valoración de Letras del Tesoro (cupón cero)

**Precio de compra:**
```
P = N / (1 + i × d/360)

Donde:
N = Nominal
i = Tipo de interés
d = Días hasta vencimiento
```

**Ejemplo:**
```
Letra a 180 días, nominal 1.000€, tipo 3%

P = 1.000 / (1 + 0,03 × 180/360)
P = 1.000 / 1,015 = 985,22€
```

**Rentabilidad de una Letra:**
```
Rentabilidad = [(N - P) / P] × (360/d) × 100
```

**Ejemplo:**
```
Compra: 985,22€, Venta/Vencimiento: 1.000€, Plazo: 180 días

Rentabilidad = [(1.000 - 985,22) / 985,22] × (360/180) × 100 = 3%
```

#### Valoración de Bonos con Cupón

**Fórmula general:**
```
P = Σ [Cupón / (1+TIR)^t] + [Nominal / (1+TIR)^n]
```

**Ejemplo detallado:**
```
Bono:
- Nominal: 1.000€
- Cupón: 5% anual
- Vencimiento: 3 años
- TIR mercado: 4%

Año 1: 50 / (1,04)^1 = 48,08€
Año 2: 50 / (1,04)^2 = 46,23€
Año 3: (50 + 1.000) / (1,04)^3 = 933,51€

Precio = 48,08 + 46,23 + 933,51 = 1.027,82€
```

**Relación Precio-TIR:**
- **TIR < Cupón:** Bono cotiza con prima (P > 100)
- **TIR = Cupón:** Bono cotiza a la par (P = 100)
- **TIR > Cupón:** Bono cotiza con descuento (P < 100)

#### Precio Entero vs. Ex-cupón

```
Precio Entero = Precio Ex-cupón + Cupón Corrido

Cupón Corrido = Cupón anual × (días desde último cupón / 365)
```

**Ejemplo:**
```
Bono con cupón 5% (50€), último cupón hace 90 días

Cupón corrido = 50 × (90/365) = 12,33€

Si Precio ex-cupón = 1.020€
Precio entero = 1.020 + 12,33 = 1.032,33€
```

### 4.7 Principios de Malkiel

1. **Relación inversa precio-TIR:** Si sube la TIR, baja el precio y viceversa.

2. **Mayor plazo = Mayor sensibilidad:** Bonos a largo plazo fluctúan más que los de corto plazo.

3. **Sensibilidad creciente a tasa decreciente:** El efecto de cambios en TIR disminuye con plazos muy largos.

4. **Asimetría:** Una bajada de TIR sube el precio MÁS de lo que una subida equivalente lo baja.

5. **Menor cupón = Mayor sensibilidad:** Bonos con cupones bajos (o cupón cero) son más sensibles.

### 4.8 ETTI y Curva de Tipos

**ETTI (Estructura Temporal de Tipos de Interés):**
Relación entre tipos de interés y plazos.

**Formas de la curva:**

| Forma | Significado |
|-------|-------------|
| **Normal (creciente)** | Expectativas de crecimiento/inflación |
| **Invertida** | Expectativas de recesión |
| **Plana** | Incertidumbre, transición |
| **Con joroba** | Expectativas de cambio a medio plazo |

**Teorías explicativas:**

| Teoría | Explicación |
|--------|-------------|
| **Expectativas** | Los forwards reflejan tipos esperados |
| **Liquidez** | Prima por invertir a largo plazo |
| **Segmentación** | Cada plazo es un mercado independiente |
| **Hábitat preferido** | Inversores prefieren ciertos plazos |

### 4.9 Duración y Sensibilidad

**Duración de Macaulay:**
Media ponderada de los vencimientos de los flujos, usando como pesos el valor actual de cada flujo.

```
D = Σ [t × VA(Flujo_t)] / Precio
```

**Duración Modificada (Corregida):**
```
DM = D / (1 + TIR)
```

**Uso de la Duración Modificada:**
```
ΔP% ≈ -DM × Δi

Donde Δi = cambio en TIR (en puntos porcentuales)
```

**Ejemplo:**
```
Bono con DM = 5 años. Si la TIR sube 0,50%:

ΔP% ≈ -5 × 0,50% = -2,50%

Si el precio era 100, el nuevo precio será aproximadamente 97,50.
```

**Propiedades de la Duración:**
- A mayor plazo, mayor duración
- A mayor cupón, menor duración
- A mayor TIR, menor duración
- Duración de cupón cero = plazo

**Sensibilidad:**
```
Sensibilidad = -DM × P / 100
```

**Duración de una cartera:**
```
D_cartera = Σ (wi × Di)

Donde wi = peso del bono i en la cartera
```

### 4.10 Inmunización

**Concepto:** Igualar la duración de activos y pasivos para protegerse del riesgo de tipo de interés.

**Ejemplo:**
```
Pasivo: Pago de 100.000€ dentro de 5 años
Objetivo: Inmunizar con bonos

Solución: Crear cartera de bonos con Duración = 5 años
```

## 5. MERCADO DE RENTA VARIABLE

### 5.1 Características

**Definición:** Participación en el capital de una empresa con derechos económicos y políticos.

**Derechos del accionista:**
- Económicos: Dividendos, suscripción preferente, cuota de liquidación
- Políticos: Voto, información, impugnación

**Fuentes de rentabilidad:**
```
Rentabilidad Total = Plusvalía + Dividendos

Rentabilidad = [(P1 - P0 + D) / P0] × 100
```

### 5.2 Estructura del Mercado Bursátil Español

**Organismos:**

| Organismo | Función |
|-----------|---------|
| **CNMV** | Supervisión y regulación |
| **BME (Bolsas y Mercados Españoles)** | Holding de mercados |
| **Sociedades Rectoras** | Gestión de cada bolsa |
| **Iberclear** | Compensación y liquidación |

**SIBE (Sistema de Interconexión Bursátil Español):**
- Plataforma electrónica de negociación
- Horario: 9:00 - 17:30

**Fases de negociación:**

| Fase | Horario | Características |
|------|---------|-----------------|
| Subasta apertura | 8:30-9:00 | Se forma precio de apertura |
| Mercado abierto | 9:00-17:30 | Negociación continua |
| Subasta cierre | 17:30-17:35 | Se forma precio de cierre |

### 5.3 Tipos de Órdenes

| Orden | Descripción |
|-------|-------------|
| **De mercado** | Se ejecuta al mejor precio disponible |
| **Limitada** | Solo se ejecuta al precio indicado o mejor |
| **Por lo mejor** | Se ejecuta al mejor precio, pero se limita |
| **Stop** | Se activa cuando se alcanza un precio |

**Condiciones de ejecución:**
- Ejecutar o anular
- Volumen mínimo
- Todo o nada

**Prioridades:** 1º Precio, 2º Tiempo

### 5.4 Operaciones Bursátiles

#### OPV (Oferta Pública de Venta)
- Venta de acciones existentes
- Los fondos van a los vendedores

#### OPS (Oferta Pública de Suscripción)
- Emisión de acciones nuevas
- Los fondos van a la empresa

#### OPA (Oferta Pública de Adquisición)
- Oferta para comprar acciones de una empresa
- Obligatoria cuando se supera el 30% del capital

#### Split
- División del nominal de las acciones
- Aumenta el número de acciones, baja el precio proporcional
- No cambia el valor total

**Ejemplo:**
```
Split 2x1: Acción de 20€ → 2 acciones de 10€
```

#### Ampliación de Capital

**Valor teórico del derecho de suscripción:**
```
DS = (P0 - Pe) / (n + 1)

Donde:
P0 = Precio antes de la ampliación
Pe = Precio de emisión
n = Número de acciones antiguas por cada nueva
```

**Ejemplo:**
```
Ampliación 1x4 (1 nueva por cada 4 antiguas)
P0 = 25€, Pe = 20€

DS = (25 - 20) / (4 + 1) = 5 / 5 = 1€

Precio teórico ex-derecho = 25 - 1 = 24€
```

#### Operaciones a Crédito

**Compra a crédito:**
```
Rentabilidad = [(Pv - Pc) / Fondos propios] × 100 - Intereses
```

**Venta a crédito (cortos):**
```
Rentabilidad = [(Pv - Pc) / Garantía depositada] × 100 - Coste préstamo títulos
```

**Ejemplo compra a crédito:**
```
Compra 100 acciones a 50€ = 5.000€
Aportación propia: 2.500€ (50%)
Préstamo: 2.500€ al 5% anual

Venta después de 1 año a 60€ = 6.000€
Plusvalía = 1.000€
Intereses = 125€
Beneficio neto = 875€

Rentabilidad = 875 / 2.500 = 35%
(vs. 20% sin apalancamiento)
```

### 5.5 Dividendos

**Tipos:**
- **Dividendo a cuenta:** Anticipo del dividendo final
- **Dividendo complementario:** Resto tras aprobar cuentas
- **Dividendo extraordinario:** Por eventos excepcionales
- **Scrip dividend:** Opción de recibir en acciones

**Rentabilidad por dividendo:**
```
Dividend Yield = (DPA / Precio) × 100

Donde DPA = Dividendo Por Acción
```

### 5.6 Índices Bursátiles

**Definición:** Indicador que refleja la evolución de un conjunto de valores.

**Métodos de cálculo:**

| Método | Descripción | Ejemplo |
|--------|-------------|---------|
| **Capitalización** | Peso según capitalización bursátil | IBEX-35, S&P 500 |
| **Ponderado por precio** | Peso según precio de la acción | Dow Jones |
| **Equiponderado** | Mismo peso para todos | S&P 500 Equal Weight |

**IBEX-35:**
- 35 valores más líquidos de la Bolsa española
- Ponderado por capitalización ajustada por free float
- **Free float:** Acciones que circulan libremente en el mercado
- **Capital cautivo:** Participaciones estables (fundadores, Estado, etc.)

**Cálculo del IBEX-35:**
```
IBEX-35 = [Σ(Capitalización ajustada) / Divisor] × 100
```

**Índices ESG españoles:**
- **FTSE4Good IBEX:** Empresas con buenas prácticas ESG
- **IBEX Gender Equality:** Empresas con igualdad de género

### 5.7 Análisis Fundamental

**Objetivo:** Determinar el valor intrínseco de una empresa.

**Métodos de valoración:**

| Método | Descripción |
|--------|-------------|
| **Valor contable** | Patrimonio neto / Nº acciones |
| **Descuento de dividendos** | VA de dividendos futuros |
| **Descuento de flujos (DCF)** | VA de flujos de caja libres |
| **Múltiplos comparables** | PER, EV/EBITDA del sector |

#### Ratios Bursátiles

**PER (Price Earnings Ratio):**
```
PER = Precio / BPA

Donde BPA = Beneficio Por Acción
```

- PER alto: Expectativas de crecimiento o sobrevaloración
- PER bajo: Infravaloración o problemas esperados

**Precio teórico según PER:**
```
Precio = PER × BPA
```

**Rentabilidad por dividendo:**
```
Div. Yield = DPA / Precio
```

**Earning Yield:**
```
Earning Yield = BPA / Precio = 1 / PER
```

**Earning Yield Gap:**
```
EYG = Earning Yield - Rentabilidad bono 10 años
```

Si EYG > 0: Renta variable más atractiva que bonos

**Precio/Valor Contable (P/VC):**
```
P/VC = Precio / Valor contable por acción
```

**ROE (Return on Equity):**
```
ROE = Beneficio Neto / Fondos Propios × 100
```

**ROA (Return on Assets):**
```
ROA = Beneficio Neto / Activo Total × 100
```

**Relación P/VC, ROE y PER:**
```
P/VC = ROE × PER
```

#### Modelo de Gordon-Shapiro

**Fórmula:**
```
P = D1 / (k - g)

Donde:
D1 = Dividendo esperado año 1
k = Rentabilidad exigida
g = Tasa de crecimiento del dividendo
```

**Cálculo de g (crecimiento sostenible):**
```
g = ROE × (1 - Pay-out)

Pay-out = DPA / BPA
```

**Ejemplo:**
```
D0 = 2€, g = 4%, k = 10%

D1 = 2 × 1,04 = 2,08€
P = 2,08 / (0,10 - 0,04) = 2,08 / 0,06 = 34,67€
```

### 5.8 Análisis Técnico

**Principios básicos:**
1. El precio lo descuenta todo
2. Los precios se mueven en tendencias
3. La historia se repite

**Soportes y Resistencias:**
- **Soporte:** Nivel donde la demanda frena las caídas
- **Resistencia:** Nivel donde la oferta frena las subidas
- Un soporte roto se convierte en resistencia y viceversa

**Tendencias:**
- **Alcista:** Mínimos y máximos crecientes
- **Bajista:** Mínimos y máximos decrecientes
- **Lateral:** Sin dirección clara

#### Figuras Chartistas

**Figuras de vuelta:**
- Hombro-Cabeza-Hombro (HCH)
- Doble techo / Doble suelo
- Triple techo / Triple suelo

**Figuras de continuación:**
- Triángulos
- Banderas y gallardetes
- Rectángulos

**Cálculo de objetivos:**
```
HCH: Objetivo = Línea clavicular - (Cabeza - Línea clavicular)
Doble techo: Objetivo = Soporte - (Máximo - Soporte)
Triángulo: Objetivo = Ruptura ± Base del triángulo
```

#### Indicadores y Osciladores

**Media Móvil (MM):**
- Suaviza el precio
- Señal de compra: Precio cruza MM al alza
- Señal de venta: Precio cruza MM a la baja

**MACD:**
- Diferencia entre MM corta y larga
- Señal de compra: MACD cruza su señal al alza
- Divergencias precio-MACD anticipan cambios de tendencia

**RSI (Relative Strength Index):**
- Oscilador entre 0 y 100
- >70: Sobrecompra
- <30: Sobreventa

**Estocástico:**
- Compara precio actual con rango de precios
- >80: Sobrecompra
- <20: Sobreventa

## 6. MERCADO DE DIVISAS

### 6.1 Conceptos Básicos

**Tipo de cambio:** Precio de una moneda en términos de otra.

**Formas de cotización:**
- **Directa:** Unidades de moneda nacional por 1 unidad de moneda extranjera (1 USD = 0,92 EUR)
- **Indirecta:** Unidades de moneda extranjera por 1 unidad de moneda nacional (1 EUR = 1,09 USD)

**Divisa base vs. cotizada:**
```
EUR/USD = 1,09

EUR = Divisa base (la que vale 1 unidad)
USD = Divisa cotizada (precio de la base)
```

**Bid-Ask en divisas:**
- **Bid:** Precio al que el banco compra la divisa base
- **Ask:** Precio al que el banco vende la divisa base
- **Spread:** Ask - Bid (beneficio del intermediario)

### 6.2 Mercado FOREX

**Características:**
- Mayor mercado del mundo (+6 billones USD/día)
- Operativo 24 horas
- Descentralizado (OTC)
- Principales pares: EUR/USD, USD/JPY, GBP/USD

### 6.3 Tipos de Cambio Cruzados

**Cálculo:**
```
EUR/GBP = EUR/USD × USD/GBP = EUR/USD ÷ GBP/USD
```

**Ejemplo:**
```
EUR/USD = 1,10
GBP/USD = 1,25

EUR/GBP = 1,10 / 1,25 = 0,88
(1 EUR = 0,88 GBP)
```

### 6.4 Formación de Precios

**Teoría de la Paridad de Tipos de Interés (PTI):**
```
Ft/S0 = (1 + id) / (1 + ie)

Donde:
Ft = Tipo forward
S0 = Tipo spot
id = Tipo de interés doméstico
ie = Tipo de interés extranjero
```

**Teoría de la Paridad del Poder Adquisitivo (PPA):**
- A largo plazo, los tipos de cambio reflejan diferencias de inflación
- Países con mayor inflación → depreciación de su moneda

### 6.5 Mercado Spot vs. Forward

**Spot:**
- Liquidación en T+2
- Tipo de cambio actual

**Forward:**
- Acuerdo para intercambio futuro
- Tipo pactado hoy, ejecución futura

**Cálculo del tipo forward:**
```
Forward = Spot × [(1 + id × t) / (1 + ie × t)]
```

**Ejemplo:**
```
Spot EUR/USD = 1,10
Tipo interés EUR = 3%
Tipo interés USD = 5%
Plazo = 1 año

Forward = 1,10 × (1,03 / 1,05) = 1,10 × 0,981 = 1,079

El EUR se aprecia frente al USD en el forward porque los tipos USD son mayores.
```

**Puntos Swap:**
```
Puntos Swap = Forward - Spot
```

Si puntos > 0: La divisa base cotiza con prima
Si puntos < 0: La divisa base cotiza con descuento

### 6.6 Riesgo de Tipo de Cambio

**Definición:** Posibilidad de pérdida por movimientos adversos en tipos de cambio.

**Ejemplo:**
```
Inversor español compra acciones USA por 10.000 USD cuando EUR/USD = 1,10
Inversión en EUR = 10.000 / 1,10 = 9.091 EUR

Un año después:
- Acciones suben 10%: 11.000 USD
- EUR/USD = 1,20 (euro se aprecia)

Valor en EUR = 11.000 / 1,20 = 9.167 EUR
Rentabilidad real = (9.167 - 9.091) / 9.091 = 0,84%

La apreciación del euro ha anulado casi toda la ganancia de la acción.
```

### 6.7 Criptodivisas (Nociones)

**Definición:** Activos digitales que usan criptografía para asegurar transacciones.

**Características:**
- Descentralizadas (blockchain)
- Oferta limitada o programada
- Alta volatilidad

**Principales por capitalización:** Bitcoin, Ethereum

## 7. MERCADO DE PRODUCTOS DERIVADOS

### 7.1 Conceptos Básicos

**Definición:** Instrumentos cuyo valor depende de otro activo (subyacente).

**Usos:**
- **Cobertura (Hedging):** Reducir riesgo
- **Especulación:** Apostar por movimientos de precio
- **Arbitraje:** Aprovechar ineficiencias de precio

**Mercado organizado vs. OTC:**

| Característica | Organizado | OTC |
|----------------|------------|-----|
| Estandarización | Sí | No |
| Cámara de compensación | Sí | No |
| Riesgo contrapartida | Bajo | Alto |
| Liquidez | Alta | Variable |
| Ejemplo | Futuros MEFF | Forwards, Swaps |

### 7.2 Futuros

**Definición:** Contrato estandarizado para comprar/vender un activo a un precio y fecha determinados.

**Características:**
- Obligación para ambas partes
- Liquidación diaria (mark-to-market)
- Requiere garantías (margen)

**Posiciones:**
- **Largo (comprador):** Se beneficia si sube el precio
- **Corto (vendedor):** Se beneficia si baja el precio

**Diferencia Futuro vs. Forward:**

| Futuro | Forward |
|--------|---------|
| Estandarizado | Personalizado |
| Mercado organizado | OTC |
| Liquidación diaria | Al vencimiento |
| Sin riesgo contrapartida | Con riesgo contrapartida |

**Precio teórico del futuro:**
```
F = S × (1 + r × t) - D

Donde:
S = Precio spot
r = Tipo de interés
t = Tiempo hasta vencimiento
D = Dividendos esperados (si aplica)
```

**Ejemplo:**
```
Acción cotiza a 100€
Tipo de interés: 3% anual
Plazo: 6 meses (0,5 años)
Dividendo esperado: 2€

F = 100 × (1 + 0,03 × 0,5) - 2 = 100 × 1,015 - 2 = 99,50€
```

**Resultado de una posición de futuros:**
```
Resultado Largo = Precio Final - Precio Entrada
Resultado Corto = Precio Entrada - Precio Final
```

**Ratio de cobertura:**
```
Nº contratos = (Valor cartera × Beta) / (Precio futuro × Multiplicador)
```

**Ejemplo cobertura:**
```
Cartera de 500.000€, Beta = 1,2
Futuro IBEX a 10.000 puntos, multiplicador = 10€

Nº contratos = (500.000 × 1,2) / (10.000 × 10) = 6 contratos cortos
```

### 7.3 Opciones

**Definición:** Derecho (no obligación) a comprar o vender un activo a un precio determinado.

**Tipos:**
- **Call:** Derecho a COMPRAR
- **Put:** Derecho a VENDER

**Posiciones básicas:**

| Posición | Expectativa | Riesgo máximo | Beneficio máximo |
|----------|-------------|---------------|------------------|
| Compra Call | Alcista | Prima pagada | Ilimitado |
| Venta Call | Bajista/Neutral | Ilimitado | Prima cobrada |
| Compra Put | Bajista | Prima pagada | Strike - Prima |
| Venta Put | Alcista/Neutral | Strike - Prima | Prima cobrada |

**Componentes de la prima:**
```
Prima = Valor Intrínseco + Valor Temporal
```

**Valor intrínseco:**
- Call: máx(0, Spot - Strike)
- Put: máx(0, Strike - Spot)

**Clasificación según valor intrínseco:**

| Estado | Call | Put |
|--------|------|-----|
| **In-The-Money (ITM)** | Spot > Strike | Spot < Strike |
| **At-The-Money (ATM)** | Spot ≈ Strike | Spot ≈ Strike |
| **Out-of-The-Money (OTM)** | Spot < Strike | Spot > Strike |

**Ejemplo Call:**
```
Call sobre acción
Strike = 50€
Prima = 3€
Spot al vencimiento = 58€

Valor intrínseco = 58 - 50 = 8€
Resultado = 8 - 3 = 5€ de beneficio

Punto de equilibrio (Break-even) = Strike + Prima = 53€
```

**Ejemplo Put:**
```
Put sobre acción
Strike = 50€
Prima = 2€
Spot al vencimiento = 45€

Valor intrínseco = 50 - 45 = 5€
Resultado = 5 - 2 = 3€ de beneficio

Punto de equilibrio = Strike - Prima = 48€
```

**Variables que afectan a la prima:**

| Variable | Efecto en Call | Efecto en Put |
|----------|---------------|---------------|
| ↑ Precio subyacente | ↑ Prima | ↓ Prima |
| ↑ Strike | ↓ Prima | ↑ Prima |
| ↑ Tiempo vencimiento | ↑ Prima | ↑ Prima |
| ↑ Volatilidad | ↑ Prima | ↑ Prima |
| ↑ Tipo interés | ↑ Prima | ↓ Prima |
| ↑ Dividendos | ↓ Prima | ↑ Prima |

**Opción europea vs. americana:**
- Europea: Solo ejercicio al vencimiento
- Americana: Ejercicio en cualquier momento

### 7.4 Swaps (Nociones)

**Definición:** Intercambio de flujos de caja entre dos partes.

**Swap de tipos de interés (IRS):**
- Una parte paga tipo fijo
- Otra parte paga tipo variable
- Se intercambian solo los flujos de intereses, no el principal

**Uso:** Gestionar exposición a tipos de interés sin modificar la deuda original.

### 7.5 Productos Estructurados

**Definición:** Combinación de renta fija + derivados para crear perfiles de riesgo/rentabilidad específicos.

**Componentes típicos:**
1. Bono cupón cero (garantía del capital)
2. Opción (participación en el subyacente)

**Ejemplo de fondo garantizado:**
```
Inversión: 100€
Garantía: 100% del capital
Participación: 50% en subida del IBEX

Construcción:
- Bono cupón cero 3 años al 3%: VA = 100/1,03³ = 91,51€
- Disponible para opciones: 100 - 91,51 = 8,49€
- Si la opción cuesta 17% → Participación = 8,49/17 ≈ 50%
```

### 7.6 Productos Complejos (MiFID)

**Definición:** Productos que requieren evaluación de idoneidad reforzada.

**Ejemplos:**
- Participaciones preferentes
- Deuda subordinada
- CoCos (bonos contingentes convertibles)
- Derivados complejos
- Estructurados sin garantía

## 8. INVERSIONES SOSTENIBLES (ESG)

### 8.1 Definiciones

**Finanzas sostenibles:** Integración de criterios ESG en decisiones financieras.

**Inversión sostenible:** Inversión que contribuye a objetivos medioambientales o sociales.

### 8.2 Estrategias de Inversión ESG

### 8.3 Marco Normativo

## Caso 1: Valoración de Bonos

**Enunciado:**
Un bono del Estado tiene las siguientes características:
- Nominal: 1.000€
- Cupón: 4% anual
- Vencimiento: 4 años
- TIR actual del mercado: 5%

**a) Calcule el precio del bono**

**Solución:**
```
Año 1: 40 / (1,05)^1 = 38,10€
Año 2: 40 / (1,05)^2 = 36,28€
Año 3: 40 / (1,05)^3 = 34,55€
Año 4: 1.040 / (1,05)^4 = 855,61€

Precio = 38,10 + 36,28 + 34,55 + 855,61 = 964,54€
```

**b) ¿Cotiza con prima o descuento?**

**Solución:** Cotiza con descuento (964,54 < 1.000) porque TIR (5%) > Cupón (4%)

**c) Si la TIR baja al 3%, ¿qué ocurre con el precio?**

**Solución:** El precio SUBE (relación inversa precio-TIR)
```
Nuevo precio = 40/1,03 + 40/1,03² + 40/1,03³ + 1.040/1,03⁴ = 1.037,17€
```

## Caso 2: Duración y Sensibilidad

**Enunciado:**
Un bono tiene duración modificada de 6 años y cotiza al 102%.

**a) Si los tipos suben 50 pb, ¿cuál será el nuevo precio aproximado?**

**Solución:**
```
ΔP% ≈ -DM × Δi = -6 × 0,50% = -3%

Precio nuevo = 102 × (1 - 0,03) = 102 × 0,97 = 98,94%
```

**b) ¿Cuánto debería mover los tipos para que el bono baje un 1,5%?**

**Solución:**
```
-1,5% = -6 × Δi
Δi = 1,5% / 6 = 0,25% = 25 pb de subida
```

## Caso 3: Tipo Forward

**Enunciado:**
La curva de tipos muestra:
- Tipo spot 1 año: 2,5%
- Tipo spot 2 años: 3,0%

**Calcule el tipo forward del año 1 al año 2.**

**Solución:**
```
(1 + r0,2)^2 = (1 + r0,1)^1 × (1 + f1,2)^1

(1,03)^2 = (1,025) × (1 + f1,2)
1,0609 = 1,025 × (1 + f1,2)
1 + f1,2 = 1,0609 / 1,025 = 1,0350
f1,2 = 3,50%
```

El mercado espera que el tipo a 1 año dentro de un año sea del 3,50%.

## Caso 4: Ampliación de Capital

**Enunciado:**
Una empresa realiza ampliación 1×5 (1 acción nueva por cada 5 antiguas):
- Precio antes de la ampliación: 30€
- Precio de emisión: 24€

**a) Calcule el valor teórico del derecho de suscripción**

**Solución:**
```
DS = (P0 - Pe) / (n + 1) = (30 - 24) / (5 + 1) = 6 / 6 = 1€
```

**b) Calcule el precio teórico ex-derecho**

**Solución:**
```
Precio ex-derecho = P0 - DS = 30 - 1 = 29€
```

## Caso 5: Opciones

**Enunciado:**
Un inversor compra una Call con:
- Strike: 100€
- Prima: 5€

**a) Resultado si al vencimiento el subyacente cotiza a 115€**

**Solución:**
```
Valor intrínseco = máx(0, 115-100) = 15€
Resultado = 15 - 5 = +10€ (beneficio)
```

**b) Resultado si al vencimiento cotiza a 98€**

**Solución:**
```
Valor intrínseco = máx(0, 98-100) = 0€
Resultado = 0 - 5 = -5€ (pérdida = prima pagada)
```

**c) Punto de equilibrio (break-even)**

**Solución:**
```
Break-even = Strike + Prima = 100 + 5 = 105€
```

## Caso 6: Tipo de Cambio Forward

**Enunciado:**
- Spot EUR/USD = 1,10
- Tipo de interés EUR = 2%
- Tipo de interés USD = 4%
- Plazo = 6 meses

**Calcule el tipo forward EUR/USD a 6 meses**

**Solución:**
```
Forward = Spot × [(1 + rEUR × t) / (1 + rUSD × t)]
Forward = 1,10 × [(1 + 0,02 × 0,5) / (1 + 0,04 × 0,5)]
Forward = 1,10 × [1,01 / 1,02]
Forward = 1,10 × 0,9902 = 1,0892

Puntos swap = 1,0892 - 1,10 = -0,0108 = -108 pips
```

El EUR cotiza con descuento en el forward porque los tipos USD son mayores.

## Resumen de Fórmulas Clave

| Concepto | Fórmula |
|----------|---------|
| Capitalización simple | Cn = C0 × (1 + i×n) |
| Capitalización compuesta | Cn = C0 × (1+i)^n |
| TAE | TAE = (1 + i/m)^m - 1 |
| Tipo forward | (1+r0,n)^n = (1+r0,m)^m × (1+fm,n)^(n-m) |
| Renta perpetua | VA = C / i |
| Duración modificada | DM = D / (1+TIR) |
| Cambio precio bono | ΔP% ≈ -DM × Δi |
| PER | PER = Precio / BPA |
| Gordon-Shapiro | P = D1 / (k - g) |
| Derecho suscripción | DS = (P0 - Pe) / (n + 1) |
| Forward divisa | F = S × [(1+rd×t) / (1+re×t)] |
